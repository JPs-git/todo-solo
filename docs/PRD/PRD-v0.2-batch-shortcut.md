## 一、功能概述

### 1.1 功能目标

为v0.2版本添加两个提升操作效率的功能：
1. **批量操作**：支持多选任务进行批量删除和批量标记完成
2. **快捷键支持**：提供键盘快捷键，提升键盘操作效率

### 1.2 用户价值

- **效率提升**：批量处理大量任务，减少重复操作
- **操作便捷**：键盘快捷键让熟练用户无需鼠标即可完成常用操作
- **用户体验**：符合专业用户的操作习惯，提升产品竞争力

### 1.3 目标用户

- 高频使用者：每天管理大量任务的职场人士
- 效率追求者：习惯使用键盘快捷键的进阶用户
- 批量处理需求：需要定期清理或归档任务的用户

## 四、交互设计

### 4.1 批量操作流程

#### 选择任务
1. 用户点击任务卡片左侧复选框
2. 任务被选中，复选框显示选中状态
3. 批量操作工具栏出现
4. 选择计数更新

#### 全选任务
1. 用户点击"全选"按钮
2. 所有可见任务被选中
3. 选择计数显示总任务数

#### 批量删除
1. 用户选中一个或多个任务
2. 用户点击"删除"按钮
3. 弹出确认对话框："确定要删除选中的 X 个任务吗？"
4. 用户确认后，删除所有选中任务
5. 显示成功提示："已删除 X 个任务"
6. 批量操作工具栏隐藏

#### 批量标记完成
1. 用户选中一个或多个任务
2. 用户点击"完成"按钮
3. 所有选中任务标记为已完成
4. 显示成功提示："已标记 X 个任务为已完成"
5. 任务列表更新，已完成任务移到底部

### 4.2 快捷键操作流程

#### 新建任务（Ctrl+N）
1. 用户按下 Ctrl+N
2. 新建任务表单弹出
3. 光标自动聚焦到标题输入框

#### 确认保存（Enter）
1. 用户在表单中填写任务信息
2. 用户按下 Enter
3. 表单提交，任务保存
4. 表单关闭，返回任务列表

#### 取消操作（Esc）
1. 用户按下 Esc
2. 如果有打开的表单/弹窗，则关闭
3. 如果处于批量选择模式，则取消所有选择

#### 聚焦搜索（Ctrl+F）
1. 用户按下 Ctrl+F
2. 光标自动聚焦到搜索框
3. 搜索框内容被选中（如果有）

#### 删除选中（Delete）
1. 用户选中一个或多个任务
2. 用户按下 Delete 键
3. 弹出确认对话框
4. 确认后删除选中任务

#### 标记完成（Space）
1. 用户选中一个或多个任务
2. 用户按下 Space 键
3. 切换选中任务的完成状态

### 4.3 快捷键冲突处理

| 场景 | 处理方式 |
|-----|---------|
| 在输入框中按快捷键 | 不触发全局快捷键，执行输入框默认行为 |
| 在表单中按Esc | 关闭表单，不触发其他操作 |
| 浏览器默认快捷键冲突 | 阻止默认行为，执行应用快捷键 |
| 多选状态下按N | 如果输入框未聚焦，触发新建任务 |

};

// 批量删除
const deleteSelectedTasks = () => {
  dispatch({ type: 'DELETE_TASKS', payload: state.selectedTaskIds });
  dispatch({ type: 'DESELECT_ALL_TASKS' });

// 批量标记完成
const completeSelectedTasks = () => {
  dispatch({ type: 'TOGGLE_TASKS', payload: state.selectedTaskIds });

### 6.2 快捷键实现

#### 使用自定义Hook
```typescript
// hooks/useKeyboardShortcuts.ts
import { useEffect } from 'react';

export const useKeyboardShortcuts = (shortcuts: Record<string, () => void>) => {
  useEffect(() => {
    const handleKeyDown = (e: KeyboardEvent) => {
      // 忽略输入框中的快捷键
      if (e.target instanceof HTMLInputElement || 
          e.target instanceof HTMLTextAreaElement) {
        return;
      }

      const key = `${e.ctrlKey || e.metaKey ? 'Ctrl+' : ''}${e.key}`;
      
      if (shortcuts[key]) {
        e.preventDefault();
        shortcuts[key]();
      }
    };

    window.addEventListener('keydown', handleKeyDown);
    return () => window.removeEventListener('keydown', handleKeyDown);
  }, [shortcuts]);
};
```

#### 在App组件中使用
```typescript
// App.tsx
useKeyboardShortcuts({
  'Ctrl+n': () => handleOpenTaskForm(),
  'Ctrl+f': () => document.getElementById('search-input')?.focus(),
  'Ctrl+a': () => selectAllTasks(),
  'Delete': () => {
    if (state.selectedTaskIds.length > 0) {
      handleBatchDelete();
    }
  },
  '?': () => setShowHelpModal(true),
});
```
